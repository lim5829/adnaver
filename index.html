🔗 Cloudflare Worker 프록시 URL
CORS 우회를 위해 프록시가 필요합니다. 아래 가이드로 1분 만에 설정 가능합니다.
📡 프록시 Worker URL 배포 후 Worker URL을 여기에 입력하세요
1

workers.cloudflare.com 에서 무료 가입 (이미 있으면 로그인)

2

Workers & Pages → Create → Hello World 템플릿 선택

3

아래 "Worker 코드 복사" 버튼 클릭 → 에디터에 붙여넣기 → Deploy

4

배포 후 나오는 *.workers.dev URL을 위 입력칸에 붙여넣기

네이버 검색광고 시스템 → 도구 → API 관리에서 발급
